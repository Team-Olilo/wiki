# Olilo Customer Guides

Customer support guides for Olilo users.

## Contributing

We welcome contributions to improve our guides!

### Guide Structure

Each guide should include:

```markdown
---
title: Guide Title
description: Brief description
category: Setup|Troubleshooting|Account|Advanced
author: Your Name
lastUpdated: DD/MM/YYYY
---

# Guide Title

Brief introduction...
```

### Writing Guidelines

- **Keep it short** - Customers don't read long guides
- **Use UK English** - fibre, optimise, centre, whilst, etc.
- **Be scannable** - Use headings, bullet points, and bold text
- **Stay practical** - Focus on actionable steps
- **Test your instructions** - Make sure they actually work

### Categories

- **Setup** - Initial configuration and installation
- **Troubleshooting** - Fixing common issues
- **Account** - Billing and account management
- **Advanced** - Technical optimisation guides

### Adding a New Guide

1. Create a new `.md` file in the `guides/` folder
2. Use the frontmatter template above
3. Keep it concise (aim for under 100 lines)
4. Use UK English spelling
5. Test all instructions
6. Submit a pull request

### Updating Existing Guides

- Update the `lastUpdated` date
- Keep changes minimal and focused
- Maintain the existing structure
- Don't make guides longer - make them clearer
